This online supplementary materials contain the R programming code and the corresponding outputs for the manuscript "PAGE: An R Package for Network Detection of Multivariate Error-Prone Gene Expression Data with The Availability of Auxiliary Information" submitted to BMC Bioinformatics.
The following list contains the files' names and their descriptions:
1. Simulation_demo_NP_Graph.R: The R script for the implementation of the function NP_Graph in simulation studies (Section 6.2).
2. Simulation_demo_Joint_Gaussian.R: The R script for the implementation of the function Joint_Gaussian in simulation studies (Section 6.2).
3. Simulation_demo_Cond_Gaussian.R: The R script for the implementation of the function Cond_Gaussian in simulation studies (Section 6.2).
4. Simulation_demo_existing_methods.R: The R script for the implementation of existing R packages in simulation studies (Section 6.2).
5. RDA_demo_NP_Graph.R: The R script for the implementation of the function NP_Graph in yeast cell cycle data analysis (Section 7.2).
6. Simulation_demo_NP_Graph_result.RData: The workspace obtained by the R script "Simulation_demo_NP_Graph.R"
7. Simulation_demo_Joint_Gaussian_result.RData: The workspace obtained by the R script "Simulation_demo_Joint_Gaussian.R"
8. Simulation_demo_Cond_Gaussian_result.RData: The workspace obtained by the R script "Simulation_demo_Cond_Gaussian.R"
9. Simulation_demo_existing_methods_result: The workspace obtained by the R script "Simulation_demo_existing_methods.R"
10. RDA_demo_NP_Graph_result: The workspace obtained by the R script "RDA_demo_NP_Grap.R"
